# Contribution à Guess-Rater

Merci de l'intérêt que vous portez au projet ! Voici comment vous pouvez aider :

## 🛠️ Installation locale
1. Forkez le repo.
2. Clonez votre fork.
3. Installez les dépendances : `npm install`.

## 🧪 Tests
Avant de proposer un changement, assurez-vous que tous les tests passent :
`npm test`

## 📬 Pull Requests
1. Créez une branche pour votre fonctionnalité.
2. Décrivez vos changements de manière claire.
3. Soumettez la PR !
